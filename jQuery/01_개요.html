jQuery 개요

jQuery란

기존에 자바 스크립트 언어만으로 구현하기에 복잡했던
구문 작성을 간소화하기 위해 개발된 Javascript 기반 라이브러리.
즉, Javascript 방식으로 작성된 유용한 함수 및 메소드가 정의되어 있음

복잡한 Javascript 구문을 보다 짧고 쉽게 개발할 수 있어 생산성이 크게 향상됨

DOM 요소와 관련한 조작 기능, 이벤트 처리 기능, AJAX 기능, 추가 확장 플러그인 등
다양한 기능을 제공함

jQuery 라이브러리 연결 방법

head 태그 내부에 script 태그로 jQuery 관련 .js 파일 가져오기
1. jQuery 라이브러리를 사이트에서 직접 다운로드 받은 후 경로를 지정하는 오프라인 방식


2. CDN(Content Delivery Network)를 통한 다운로드 방식(온라인 방식) *선호 방식
 - jQuery 라이브러리를 제공하는 사이트 url을 src 연결 경로로 지정

jQuery 구문 작성


jQuery 구문으로 DOM 요소를 다루는 구문을 작성할 예정이며,
jQuery 명령어들은 문서 상의 모든 html 요소들이 로드된 후에 실행되어야 함
    

 - 순수 Javascript로 html 페이지가 모두 로드된 이후에 코드가 실행되게 하려면
    window.onload = function(){
        // 실행 내용
    }
 - jQuery에서의 onload 함수
    1) jQuery(document).ready(function(){
         // 실행 내용
       }); // 1회만 실행됨
    2) $(document).ready(function(){
         // 실행 내용
       }); // 다중 실행 가능
    3) $(function(){
         // 실행 내용
       });

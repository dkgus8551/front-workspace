요소의 여백과 관련된 속성

        HTML에서 각 요소가 가지는 영역은총 4가지 영역으로 구성되어 있음

        1. content 영역
         - 요소의 내용이 작성되는 영역
         - 시작 태그와 종료 태그 사이의 값이 저장되는 영역
        
        2. padding 영역
         - content 영역과 border 영역 사이에 존재하는 영역
         - 요소의 여백을 표현할 때 사용

        3. border 영역
         - 요소의 테두리가 지정되는 영역
         - content 영역보다 바깥에 존재, content를 감싸는 역할

         4. margin 영역
          - 다른 요소와 현재 요소의 간격을 나타내는 영역

box-sizing 속성

         요소 영역 중 화면 상에 보여지는 부분인 content, padding, border는
        각각 별도로 크기를 지정할 수 있음
         다만, 각 속성의 크기를 따로 지정하다 보니
        전체적인 크기를 한눈에 파악하기 힘듦
         box-sizing: 요소의 크기를 contentm padding, border를
         모두 합친 크기로 지정 가능

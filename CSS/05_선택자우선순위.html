선택자 우선 순위

        css 속성은 style 태그 또는 css 파일에 작성된 순서에 맞춰서 스타일이 적용되지만,
        같은 요소에 동일한 css 속성이 설정되는 경우 우선순위 적용

        1) 동일한 우선 순위를 가진 경우, 제일 마지막에 작성된 css 속성이 반영됨
        2) 여러 선택자를 이용하여 같은 요소를 선택하더라도 지정한 css 속성이 서로 다르다면 모두 반영됨

        css 우선 순위 (명시도 순)
        1순위 : css 속성에 !important;가 포함된 경우
        2순위 : 인라인 스타일 속성
        3순위 : 아이디 선택자
        4순위 : 클래스 선택자
        5순위 : 태그 선택자
        ...

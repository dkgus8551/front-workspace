자바스크립트 개요

        Javascript
         - html의 조작과 변경을 다마당하는 언어
         - html로 웹 문서의 틀을 만들고, css로 디자인을 정의하며,
           js로 웹페이지 상의 이벤트가 발생했을 때 실행할 동작을 정의함
         - 웹 브라우저에서 실행 가능한 인터프리터 방식의 스크립트 언어
         * 인터프리터: 컴파일 없이 한 줄씩 실행되는 언어
         * Java는 .class 파일로 컴파일->JVM에서 한 줄씩 실행됨(하이브리드 언어)
        
        장점
         - 코드 작성 후 바로 실행 결과를 볼 수 있기 때문에 간단한 개발에 용이함
         - 클래스, 접근제한자, 반환형 등 고급 개념 없이 사용 가능 -> 초보자가 다루기 쉬움
         - 웹 브라우저의 기본 스크립트 언어로 범용성이 넓음

        단점
         - 컴파일 방식의 언어보다 실행 속도가 느림
         - 큰 규모의 어플리케이션 개발에 있어 "타입 안전성"이 부족함(변수 별 자료형이 일관되지 않음)

1. 자바스크립트 사용방법

  1. inline 방식 : 이벤트를 부여하고자 하는 요소 내에 간단한 소스코드를 작성해서 실행하는 방법
  2. internal 방식 : 현재 문서 내부에서 script 태그 영역에 소스 코드를 작성하여 실행하는 방법
  3. external 방식 : 현재 문서 외부에서 별도의 js 파일을 생성한 후 가져다 실행하는 방식
